# 2026-03-27 UI风格改版与功能优化

## 变更内容

### 1. 页面风格改版
- 采用浅色撞色设计风格
- 主色调：靛蓝 `#4f46e5`（按钮、表头、高亮）
- 涨跌色：红 `#dc2626` / 绿 `#16a34a`
- 背景：浅灰白 `#fafafa`
- 卡片：白色圆角设计

### 2. 组件优化

#### Toolbar 工具栏
- 添加「看自己 | 看全部」切换按钮组
- 输入框加长（搜索 160px，密钥 100px）
- 按钮统一圆角 8px，靛蓝色调

#### FundRow 基金行
- 全表粗体显示（font-weight: 600）
- 统一字体大小 14px
- 建议/原因保持持仓/空仓多行显示

#### IndexStrip 指数条
- 改为白色卡片风格
- 字体加粗、颜色加深更易读

#### FundManageModal 管理基金弹窗
- 全新卡片式布局
- 基金列表独立卡片，带悬停效果
- 添加基金实时显示名称
- 圆角按钮与整体风格统一

### 3. Bug 修复
- 修复「看全部」清空密钥输入框的问题
- 修复管理基金列表不显示基金名称的问题
- 修复点击切换视图时密钥状态不同步的问题

## 涉及文件

- `src/assets/style.css` - 全局样式改版
- `src/components/Toolbar.vue` - 工具栏优化
- `src/components/FundRow.vue` - 表格行样式
- `src/components/IndexStrip.vue` - 指数条样式
- `src/components/FundManageModal.vue` - 管理弹窗重构
- `src/components/Header.vue` - 头部简化
