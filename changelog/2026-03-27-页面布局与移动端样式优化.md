# 页面布局与移动端样式优化

## 变更概述

本次更新优化了页面布局和移动端显示效果：
1. Header + IndexStrip 包装为 sticky-header，固定在顶部
2. 增强 768px/480px 断点的响应式样式
3. 优化指数条、工具栏、表格在移动端的显示

## 文件变更

| 文件 | 变更类型 | 说明 |
|------|----------|------|
| `src/App.vue` | 修改 | 包装 sticky-header，添加移动端响应式样式 |
| `src/assets/style.css` | 修改 | 移除 index-strip 全局样式，简化响应式规则 |
| `src/components/IndexStrip.vue` | 修改 | 添加 scoped 响应式样式 |
| `src/components/Toolbar.vue` | 修改 | 添加移动端响应式样式 |
| `README.md` | 修改 | 添加 v2.0.1 版本记录 |

## 代码详情

### sticky-header 布局

```vue
<div class="sticky-header">
  <Header />
  <IndexStrip />
</div>
```

```css
.sticky-header {
  position: sticky;
  top: 0;
  z-index: 100;
}
```

### 移动端响应式

- **768px 断点**：减小 padding、字体、卡片尺寸
- **480px 断点**：进一步压缩布局，适配小屏手机
- 指数条支持横向滚动，避免内容溢出
- Toolbar 各元素缩小间距和尺寸
